# Pitch Docs

Owner: Product / Pitch Lead

This folder is for roadshow scripts, demo narration, and deck outlines.

## Pitch Spine

```text
Pain:
Traders have ideas but cannot code, backtest, or control risk.

Solution:
ProfitPrince turns natural language into a reproducible HTX strategy workflow.

Proof:
Strategy JSON, backtest metrics, simulated order logs, and hashes.

HTX Fit:
HTX pairs, HTX market data, future HTX trading API, and `$HTX` strategy marketplace potential.
```

## Demo Script Target

The full demo should fit in 3-5 minutes:

1. Natural-language input.
2. Strategy generation.
3. Backtest and benchmark.
4. AI risk explanation.
5. Simulated HTX execution.
6. Proof hash and future ecosystem plan.
