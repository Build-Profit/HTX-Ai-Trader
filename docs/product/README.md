## Current Product Direction

ProfitPrince is an HTX-native AI trading strategy agent. The MVP should prove one complete workflow:

```text
Natural-language strategy idea
-> structured strategy JSON
-> HTX market data / cached snapshot / local fallback
-> reproducible backtest
-> AI risk explanation
-> simulated execution ledger
-> proof hash
```

## Copywriting Boundary

Use:

- Strategy assistant.
- Backtest validation.
- Risk explanation.
- Simulated execution.
- Verifiable strategy record.

Avoid:

- Guaranteed profit.
- Stable profit.
- Risk-free arbitrage.
- Unverifiable fixed win rates.
- "Global first" unless supported by evidence.
